# HRMS Assessment Project

A full-stack HR Management System built with Django REST Framework and React.

## Live Demo
- **Frontend**: https://hrms-frontend.onrender.com
- **Backend API**: https://hrms-backend.onrender.com
- **Admin Panel**: https://hrms-backend.onrender.com/admin

## Features
- Employee management
- Attendance tracking
- Department management
- RESTful API
- Responsive React frontend

## Local Development

### Prerequisites
- Docker and Docker Compose
- Python 3.11+
- Node.js 18+

### Running with Docker
```bash
docker-compose up --build
